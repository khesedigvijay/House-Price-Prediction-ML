# House Price Prediction using Machine Learning

##  Project Overview
This project predicts house prices based on features like area, bedrooms,
bathrooms, and location using Machine Learning regression techniques.

## Dataset
- Source:sklearn inbuilt dataset (california_housing_dataset)
- Features: HouseAge, AveRooms, AveBedrms, Population, AveOccup, Latitude, Longitude
- Target: House Price

 ## Algorithm Used
- XGBRegressor, XGBoost (Extreme Gradient Boosting) library.
- Train-Test Split
- Model Evaluation using R² Score, MAE

 ## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Google Colab

## Results
- Achieved R² Score: 0.8338000331788725
- Low prediction error on test data
